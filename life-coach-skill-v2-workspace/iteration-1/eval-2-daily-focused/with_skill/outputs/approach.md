# Approach Notes — eval-2-daily-focused (with_skill)

## 의도 확인 (코칭 시작)

SKILL.md "코칭 시작 — 의도 확인" 규칙 적용:
- 사용자 요청: "오늘 건강이랑 과작업 부분만 봐줘"
- → "X 부분만 봐줘" 패턴 → 해당 섹션만 깊게 분석

## 선택한 섹션

coaching-prompts.md 전체 프레임(8개 섹션) 중 2개만 적용:
1. `🔍 코칭` — 과작업 패턴 분석
2. `💊 건강` — check_in, exercises, meals 요약

생략한 섹션: 오늘의 정리, 레포별 상세, 집중도 지표, 자동화 제안, 내일 이어할 것, 유통기한
→ 이유: 사용자가 명시적으로 요청하지 않은 섹션, 최소 범위 원칙 적용

마무리 질문(💬)은 포함: 단 1개 필수 규칙이므로 유지.

## 데이터 해석

**과작업 판단 근거:**
- work_hours=9.5h → 과작업 기준(8h+) 초과
- 핵심: 단순 시간 초과가 아니라 수면 5.5h + 스트레스 7 + 아침 거름이 동반된 상태에서의 과작업임을 연결해 분석

**에스컬레이션 레벨:**
- 데이터가 1일치만 제공됨 → 연속성 불명 → Level 0 적용 (부드러운 넛지)
- 단, "21시 이후 작업 끊어라"는 직접 권고 포함 (데이터 복합 악화 상태 반영)

**건강 각 항목별 기준 적용:**
- 수면: 권장 7–8h 대비 5.5h → 부족 명시
- 걸음수: 최소 5,000보 기준 → 3,000보는 절반
- 스트레스: 7/10은 높음으로 판단
- 수분: 권장 2,000ml 대비 1,200ml → 60%
- 운동: 없음 → 명시 (생략하지 않음, 프롬프트 지시사항)
- 식사: 2끼 + 아침 거름 → 영향 설명 + 내일 개선 제안

## 톤

- Level 0 기조 유지 (질문형 + 넛지)
- 단, 복합 악화 신호(수면부족+과작업+고스트레스)가 겹친 날이므로 일부 직접적 지시 포함
- 번역체·아첨·상투어 없음
