# Pantry Manager (식재료 관리)

식재료 재고 관리, 유통기한 추적, 레시피 추천을 자동화하는 스킬입니다.

## 주요 기능

- 🥬 **식재료 등록/관리** - Notion DB에 체계적으로 관리
- 📅 **유통기한 추적** - 3일 전 알림, 만료 표시
- 🍳 **레시피 추천** - 현재 재료로 만들 수 있는 저속노화 메뉴
- 🛒 **장보기 목록** - 부족한 재료 자동 감지
- 📸 **이미지 인식** - 장바구니 스크린샷 자동 파싱

## 사용법

자세한 사용법은 [SKILL.md](SKILL.md)를 참조하세요.

## 자동화

- 매일 아침 9시: 유통기한 임박 알림
- 주 1회 (일요일 저녁): 냉장고 정리 체크
