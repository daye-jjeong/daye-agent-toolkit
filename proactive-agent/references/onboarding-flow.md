# Onboarding Flow Reference

> clawd에서는 USER.md를 직접 관리하므로 이 플로우는 참조용.
> 새 사용자/에이전트 셋업 시 참고할 수 있음.

## 모드

| 모드 | 설명 |
|------|------|
| Interactive | 12개 질문을 한 번에 (~10분) |
| Drip | 세션마다 1-2개 자연스럽게 |
| Skip | 바로 시작, 대화에서 학습 |

## 기회적 학습 (Opportunistic Learning)

대화에서 자연스럽게 파악:

| 사용자 발언 | 학습 내용 |
|------------|----------|
| "나 서울에 있어" | 타임존: Asia/Seoul |
| "긴 메일 싫어" | 커뮤니케이션: 간결 |
| "공동창업자 민수가..." | 핵심 인물: 민수 (공동창업자) |
| "지금 앱 만들고 있어" | 현재 프로젝트 |

→ USER.md 업데이트, MEMORY.md에 반영

## 완료 기준

최소: 이름, 주요 목표, 커뮤니케이션 스타일
이상적: 전체 질문 완료, USER.md 완성
현실: 대부분 비공식적으로 점진적 학습
