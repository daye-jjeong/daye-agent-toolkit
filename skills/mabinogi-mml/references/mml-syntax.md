# 마비노기 모바일 MML 문법 레퍼런스

출처: [마비노기 MML연구소 (mabicompose.com)](https://www.mabicompose.com/)

---

## 음표 · 임시표

| 기호 | 설명 |
|------|------|
| `c d e f g a b` | 도~시 음표 (대소문자 무관) |
| `+` 또는 `#` | 샤프 (음표 바로 뒤) |
| `-` | 플랫 (음표 바로 뒤) |

예: `c+` (C샤프), `d-` (D플랫)

---

## 길이 · 점음표

음표 또는 쉼표 뒤에 숫자로 길이를 지정한다.

| 값 | 설명 |
|----|------|
| `1` | 온음표 |
| `2` | 2분음표 |
| `4` | 4분음표 |
| `8` | 8분음표 |
| `16` | 16분음표 |
| `32` | 32분음표 |
| `64` | 64분음표 |

점음표: 음표값 뒤에 `.` 추가 → 1.5배 길이. 중첩 가능(`c4..` = 4분음표 × 1.75).

---

## l — 기본 길이

```
l<n>
```

이후 길이를 생략한 음표/쉼표에 적용되는 기본 길이. 마비노기 기본값은 `l4`(4분음표).

예: `l8 cde` → c, d, e 모두 8분음표.

---

## 옥타브

| 기호 | 설명 |
|------|------|
| `o<n>` | 절대 옥타브 지정 (기본 `o4`) |
| `<` | 옥타브 올림 (마비노기 관례: `<`가 UP) |
| `>` | 옥타브 내림 (마비노기 관례: `>`가 DOWN) |

> **주의**: 일반 MML 관례와 `<`/`>` 방향이 반대인 구현이 있다. 마비노기 모바일 및 스크립트는 **`<` = 올림, `>` = 내림** 으로 동작한다.

**MIDI 매핑**: MIDI 60(C4) = 마비노기 `o4 c`

## 템포 · 볼륨 · 쉼표

| 명령 | 설명 |
|------|------|
| `t<n>` | 템포(BPM) 지정. 기본 `t120` |
| `v<n>` | 볼륨 지정. 기본 `v8` |
| `r<len>` | 쉼표. 길이 생략 시 현재 기본길이(`l`) 적용 |

---

## N명령 — 절대 음높이

```
n<번호>[길이][.]
```

- **음표와 동일하게 동작**: 길이를 명시하거나, 생략 시 현재 기본길이(`l`)가 적용된다.
- 옥타브 명령(`o`, `<`, `>`) 없이 절대 음번호로 음을 지정 → 글자수 절약에 유리.
- 마비꼬 export 시 "N 명령 허용" 체크박스를 활성화해야 출력에 포함된다.

---

## 멀티트랙 MML@ 포맷

```
MML@<트랙1>,<트랙2>,...,<트랙6>;
```

- 트랙은 콤마(`,`)로 구분, 전체를 세미콜론(`;`)으로 종료.
- **최대 6트랙(6화음)**. 초과 시 게임/마비꼬에서 오류.
- 각 트랙은 독립적인 MML 문자열.

예:
```
MML@t120l8cdef,t120l8efga;
```

|---------|--------|
| 옥타브 (`o`) | 4 |

| 볼륨 (`v`) | 8 |

## PC판과 차이

마비노기 **모바일**은 PC판과 악보 구조·연주 방식이 완전히 다르다 — PC용 악보를 그대로 붙여넣으면 사용 불가.
