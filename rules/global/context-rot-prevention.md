# Context Rot 방지

## 룰 파일 크기
글로벌 룰(`rules/`) 합산 6,000자 / 12K 토큰. 프로젝트 CLAUDE.md는 별도 10,000자.
넘으면 다이어트 제안. 세션 시작 시 확인 X, 룰 수정할 때만 체크.

## /compact 타이밍
구현 중 compact 금지 — 변수명, 파일 경로, 부분 상태를 잃는다.
마일스톤 전환 시(리서치→구현, 디버깅→다음 기능) compact 제안.

## MCP 서버
프로젝트당 MCP 10개 이하. 도구 설명이 매 턴 토큰을 먹는다.
안 쓰는 MCP는 disabled: true.

## 문서 분산 금지
같은 정보 여러 파일에 흩지 마라. 한 정보 = 한 파일, 다른 곳은 링크/요약만.
한 곳만 고치고 다른 곳 stale → 일관성 깨짐.
