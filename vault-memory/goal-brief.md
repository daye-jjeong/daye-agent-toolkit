# vault-memory:goal-brief

> 현재 활성 목표의 현황을 빠르게 브리핑한다.

**YAML 포맷**: goal-planner 스킬의 `references/yaml-formats.md` 참조

## 트리거

- "목표 현황", "goal status", "이번달 목표 어때"
- "이번주 진행률", "오늘 뭐 했지"
- 세션 시작 시 resume과 함께 자동 실행 가능

## 파라미터

| 형식 | 예시 | 동작 |
|------|------|------|
| (없음) | `goal-brief` | 현재 월간 + 주간 + 오늘 전체 |
| 계층 | `goal-brief monthly` | 해당 계층만 |
| 기간 | `goal-brief 2026-W07` | 특정 기간 상세 |

## 워크플로우

### 1. 파일 스캔

| 계층 | 경로 | 현재 기간 판정 |
|------|------|---------------|
| 월간 | `memory/goals/monthly/YYYY-MM.yml` | 이번 달 |
| 주간 | `memory/goals/weekly/YYYY-Www.yml` | 이번 주 |
| 일간 | `memory/goals/daily/YYYY-MM-DD.yml` | 오늘 |

`status: active` 또는 `status: completed`(회고 포함) 파일만 대상.

### 2. 전체 브리핑 (기본)

```markdown
## 목표 브리핑

### 월간: 2026-02 — "체계 잡기 + Pre-A 준비"
| 목표 | 프로젝트 | 우선순위 | KR 진행률 |
|------|----------|----------|-----------|
| SOT 체계 구축 | mingming-ai | high | 70% |
| 캘리봇 실무 적용 | ronik | high | 10% |
| 운동 루틴 정착 | health | medium | 25% |

### 주간: W07 (02/09 ~ 02/15)
| 목표 | 상태 |
|------|------|
| 캘리봇 기획 완성 | 🔄 진행중 |
| 대시보드 v3 + 목표 연동 | ✅ 완료 |
| goal-planner 크론 연동 | ⏳ todo |

### 오늘: 02/12 (수)
**Top 3:**
1. ⏳ 캘리봇 기획 마무리
2. ⏳ PM봇 기능 정의 시작
3. ✅ 스킬 정리 완료

**체크리스트:** 2/5 완료 (40%)
```

### 3. 상세 모드 (계층/기간 지정 시)

월간 상세 — KR별 current/target + 연결 태스크:
```markdown
## 2026-02 월간 목표 상세

### SOT 체계 구축 (mingming-ai) — 70%
| KR | 목표 | 현재 |
|----|------|------|
| projects/ 구조 + 규칙 | 100% | 100% ✅ |
| 대시보드 v3 | 100% | 100% ✅ |
| 크론 자동화 | 100% | 0% |

**연결 태스크:** t-ming-001 (in_progress), t-ming-005 (todo)
```

### 4. 태스크 ↔ 목표 크로스 참조

목표에 `project` 필드가 있으면 해당 프로젝트의 tasks.yml에서:
- in_progress/blocked 태스크 수
- 마감 임박 태스크
- 최근 progress_log

```markdown
> 💡 ronik 프로젝트: 6개 태스크 중 1개 진행중, 5개 대기.
> t-ronik-004 (PM봇 형식 검증) 오늘 마감.
```

### 5. 경고 표시

| 조건 | 표시 |
|------|------|
| KR current가 기간 대비 뒤처짐 | ⚠️ 진행률 주의 |
| 주간 목표 중 미시작 + 마감 3일 이내 | ⚠️ 마감 임박 |
| 일간 체크리스트 완료율 < 50% (18시 이후) | ⚠️ 오늘 미완료 |
| 회고 미작성 (기간 종료 후) | 📝 회고 필요 |

## 주의사항

- goals/*.yml을 읽기만 함 (수정 안 함)
- 완료된 목표(completed)는 기본 표시 안 함 (옵션 `--all`)
- 파일이 없는 기간은 "목표 미작성" 표시 + goal-create 제안
